ModelJS Demo

Edit, save and reload the page. Everything is stored in your browser!

This is a sample of how you can use ModelJS to work with ORM-like localStorage, to prototype your application while your API is not ready. Check the docs or get started.

The schema used in this sample app is defined by the code:

var schema = {
  _Base: {
    attrs: ['name']
  },
  Country: {
    attrs: ['abbr']
  },
  State: {
    attrs: ['abbr'],
    relsToOne: ['Country']
  },
  City: {
    relsToOne: ['State']
  }
};
